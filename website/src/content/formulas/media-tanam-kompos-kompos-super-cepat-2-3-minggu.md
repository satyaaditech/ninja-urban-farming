# Kompos Super Cepat (2-3 Minggu)

**Kategori:** MEDIA TANAM & KOMPOS  
**Nomor:** 11.2

**Bahan:**
- Mayoritas daun mangga kering (karbon cokelat)
- Sampah dapur basah: sisa sayur, kulit buah, sisa nasi (nitrogen hijau)
- Mesin pencacah
- Tong komposter aerob berlubang
- Eco-Enzyme 1:1000

**Prosedur:**
1. Cacah daun + sampah dapur.
2. Masukkan ke tong aerob, jangan padatkan.
3. Semprot EE 1:1000 merata.
4. Aduk setiap 3-4 hari.
5. **2-3 minggu:** siap pakai.

**Indikator siap:** Cokelat gelap, bau tanah hutan segar, tidak panas, gembur.
