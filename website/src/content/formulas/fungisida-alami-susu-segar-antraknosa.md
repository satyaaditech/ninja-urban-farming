# Susu Segar (Antraknosa)

**Kategori:** FUNGISIDA ALAMI  
**Nomor:** 9.1

- **Bahan:** 200 ml susu sapi segar murni + 800 ml air (rasio 1:5)
- **Cara:** Campur, langsung semprot. Jangan pakai susu bubuk/UHT.
- **Target:** Antraknosa (bercak cokelat melingkar di buah cabai/tomat)
- **Waktu:** Sore hari (butuh sinar besok untuk aktivasi)
- **Frekuensi:** 3 hari sekali

🔬 Protein whey + kasein + sinar UV → radikal bebas oksigen → hancurkan dinding sel *Colletotrichum*. Plus Kalsium → perkuat dinding sel buah.
