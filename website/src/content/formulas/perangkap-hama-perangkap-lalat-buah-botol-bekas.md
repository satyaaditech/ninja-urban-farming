# Perangkap Lalat Buah (Botol Bekas)

**Kategori:** PERANGKAP HAMA  
**Nomor:** 10.2

1. Potong bagian atas botol 600 ml, balik seperti corong.
2. Tuang 100 ml cuka dapur + 1 sdm gula + 1-2 tetes sabun.
3. Gantung dekat tanaman buah.
4. Lalat buah tertarik aroma fermentasi → masuk → tidak bisa keluar → tenggelam.
