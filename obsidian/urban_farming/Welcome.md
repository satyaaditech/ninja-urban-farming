Buka **[Urban Farming](1_Atlas/Urban%20Farming.md)** — peta konten (MOC) utama yang menghubungkan seluruh catatan riset Anda.

## 📂 Struktur Vault

| Folder | Fungsi |
|--------|--------|
| `0_Inbox/` | Catatan sementara, ide cepat, belum terproses |
| `1_Atlas/` | **Maps of Content (MOC)** — pusat navigasi antar topik |
| `2_Permanent/` | Catatan permanen, sudah matang & saling terhubung |
| `3_Literature/` | Catatan dari sumber eksternal (buku, artikel, jurnal) |
| `4_Daily/` | Catatan harian riset |
| `_Templates/` | Template untuk berbagai jenis catatan |
| `_Assets/` | Gambar, PDF, lampiran |

## 🔗 Cara Kerja Linking

- Gunakan `[[wikilink]]` untuk menghubungkan catatan — misalnya `[[Hidroponik]]`
- Setiap halaman MOC berisi daftar tautan ke subtopik
- Klik tautan redup untuk **membuat catatan baru**
- Gunakan **Graph View** (`Ctrl+G`) untuk melihat peta koneksi visual
- Gunakan panel **Backlink** untuk melihat catatan apa yang merujuk ke halaman ini

## 📝 Alur Kerja

1. **Tangkap** → Tulis ide/riset di `0_Inbox/`
2. **Proses** → Pindahkan ke `2_Permanent/` setelah matang
3. **Hubungkan** → Tambahkan `[[wikilink]]` ke catatan terkait
4. **Navigasi** → Gunakan MOC di `1_Atlas/` untuk menjelajah
