# Ekonomi Sirkular

> [!abstract] Ringkasan
> Ekonomi sirkular adalah model ekonomi di mana **tidak ada yang menjadi sampah** — setiap output dari satu proses menjadi input untuk proses lain, meniru cara kerja ekosistem alami. Dalam konteks Ninja Urban Farming, ekonomi sirkular adalah fondasi filosofis dari seluruh sistem: sampah dapur bukan dibuang, tapi menjadi pakan maggot; kotoran lele bukan limbah, tapi pupuk tanaman; daun mangga kering bukan sampah halaman, tapi mulsa dan kompos. Circular economy di skala rumah tangga bukan wacana — ia bisa dijalankan hari ini di halaman belakang.

## Isi

### Linear vs Sirkular: Dua Model Ekonomi

#### Model Linear (Ambil → Pakai → Buang)

```
AMBIL dari alam → PRODUKSI → KONSUMSI → BUANG ke TPA
     ↑                                            ↓
  [Sumber daya terkuras]                  [Sampah menumpuk]
```

Ini adalah model ekonomi dominan saat ini. Masalahnya:
- Sumber daya alam terbatas — suatu saat habis
- Tempat Pembuangan Akhir (TPA) terbatas — suatu saat penuh
- Di antara "ambil" dan "buang", nilai material hilang selamanya
- Setiap tahap menghasilkan emisi karbon

#### Model Sirkular (Ambil → Pakai → Kembalikan → Ambil Lagi)

```
        ┌──────────────────────────────────┐
        ↓                                  │
  [SUMBER DAYA] → PRODUKSI → KONSUMSI → "SAMPAH" → [DIOLAH] → [SUMBER DAYA LAGI]
```

Tidak ada terminal "buang." Setiap ujung adalah awal dari siklus baru.

### Penerapan Ekonomi Sirkular di Sistem Ninja Urban Farming

Sistem Ninja adalah implementasi ekonomi sirkular paling konkret di skala rumah tangga:

```
┌─────────────────────────────────────────────────────────┐
│                   NINJA CIRCULAR LOOP                    │
│                                                         │
│  DAPUR          → sampah organik → BIOPOND BSF          │
│    ↑                                  ↓                 │
│    │                            maggot (protein)         │
│    │                                  ↓                 │
│    │                     ┌────────────┴──────────┐      │
│    │                     ↓                       ↓      │
│    │              LELE BIOFLOK             AYAM DEEP     │
│    │              (3 galon)                LITTER         │
│    │                     ↓                       ↓      │
│    │           air kaya N (flush)    kompos (6 bln)      │
│    │                     ↓                       ↓      │
│    │                     └────────────┬──────────┘      │
│    │                                  ↓                 │
│    │                         RAISED BED                  │
│    │                    (kangkung, sawi,                 │
│    │                    tomat, cabai)                    │
│    │                                  ↓                 │
│    └──────────── SAYURAN + TELUR + LELE ────────────────┘
│                                                         │
│  Input eksternal minimal:                                │
│  • Pelet lele (berkurang seiring maggot tersedia)       │
│  • Pakan ayam 50% (berkurang seiring maggot tersedia)   │
│  • Benih (sekali beli, bisa produksi benih sendiri)     │
│  • Air (bisa pakai air hujan yang ditampung)            │
│                                                         │
│  Output yang KELUAR dari sistem:                        │
│  • MAKANAN untuk keluarga (sayur, telur, ikan)          │
│  • Kompos berlebih (bisa dijual / dibagi ke tetangga)   │
│  • Maggot berlebih (bisa dijual ke sesama peternak)     │
│  • Eco-Enzyme berlebih (bisa dijual / dibagi)           │
└─────────────────────────────────────────────────────────┘
```

### Neraca Material: Apa yang Masuk dan Keluar?

**Per Bulan (Sistem 75 m², 3 galon lele, 5 ayam):**

| INPUT (dari luar sistem) | Jumlah | Nilai (Rp) |
|--------------------------|--------|------------|
| Pelet lele (starter) | 2-3 kg | 30.000 - 45.000 |
| Pakan ayam layer | 4-5 kg | 35.000 - 50.000 |
| Benih sayur | 1-2 bungkus | 20.000 - 30.000 |
| Air | "Gratis" (sumur/PDAM) | ~5.000 |
| Listrik aerator 18W | 13 kWh/bulan | ~20.000 |
| **Total input** | | **~110.000 - 150.000** |

| OUTPUT (dikonsumsi/digunakan) | Jumlah | Setara Nilai (Rp) |
|-------------------------------|--------|-------------------|
| Sayuran daun | 4-6 kg | 80.000 - 120.000 |
| Tomat/cabai | 2-3 kg | 60.000 - 100.000 |
| Telur ayam | 20-25 butir | 50.000 - 75.000 |
| Lele konsumsi | 3-5 kg (per 3 bulan) | 75.000 - 125.000 |
| Kompos (tidak perlu beli) | 10-20 kg | ~30.000 - 60.000 |
| **Total output** | | **~295.000 - 480.000** |

> [!info] Apakah Ini "Untung"?
> **Net value: ~Rp 145.000 - 370.000/bulan** — belum termasuk nilai yang sulit diukur: sayuran bebas pestisida, telur dari ayam bebas antibiotik, ketenangan pikiran, edukasi anak-anak tentang pangan, dan ketahanan pangan keluarga.

### Riak Ekonomi Sirkular — Dampak Melampaui Kebun Sendiri

Ekonomi sirkular skala rumah tangga tidak hanya menguntungkan kamu. Ia menciptakan **riak positif** ke sistem yang lebih besar:

| Riak | Dampak ke Sistem yang Lebih Besar |
|------|----------------------------------|
| **Sampah organik tidak ke TPA** | Memperpanjang umur TPA, mengurangi emisi metana |
| **Tidak beli pupuk kimia** | Mengurangi permintaan produksi urea (5% emisi global) |
| **Tidak beli pestisida** | Mengurangi limpasan kimia ke sungai dan air tanah |
| **Tidak beli sayur dari pasar** | Mengurangi food miles dan kemasan plastik |
| **Produksi protein sendiri** | Mengurangi permintaan daging dari peternakan intensif |
| **Surplus dibagi ke tetangga** | Menularkan kesadaran pangan, menciptakan komunitas |

### Dari Circular Economy Rumah Tangga ke Circular Community

Jika satu gang/kampung/RT mengadopsi sistem serupa:

1. **Bank Sampah Organik** — Setiap rumah mengumpulkan sampah dapur → satu biopond BSF komunal → maggot dibagi ke semua pemilik lele/ayam
2. **Tukar Komoditas** — Keluarga A panen kangkung berlebih, keluarga B panen telur berlebih → tukar
3. **Sharing Tools** — Mesin pencacah, aerator cadangan, sprayer — dibeli bersama, dipakai bergiliran
4. **Edukasi** — Yang sudah berhasil mengajari yang baru mulai

### Mengukur Sirkularitas Kebun Kamu

Gunakan indikator sederhana ini:

| Indikator | Target Sirkular | Cara Mengukur |
|-----------|----------------|---------------|
| % sampah dapur yang dikomposkan | 80-100% | Timbang sampah dapur harian vs yang masuk biopond/komposter |
| % nutrisi tanaman dari sumber internal | 70-100% | Air lele + kompos ayam / total nutrisi yang diberikan |
| % pakan ternak dari sumber internal | 30-50% | Maggot + sayur afkir / total pakan |
| % air dari sumber daur ulang | 50-100% | Air hujan ditampung + air bekas cuci beras / total air digunakan |

> [!tip] Target Bertahap
> Bulan 1-3: Fokus % sampah dapur ke kompos (target 80%)
> Bulan 4-6: Fokus % nutrisi internal (target 70%)
> Bulan 7-12: Fokus % pakan internal — maggot menggantikan 30%+ pakan pabrikan
